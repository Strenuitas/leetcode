
# 思路
遍历 通过设置前结点，奇数链表删掉偶数结点同时实现当前偶结点通过尾插法创建一个新的偶链表，前结点接到下一节点
```
public static ListNode oddEvenList(ListNode head){
        ListNode oddTemp = head;
        int count = 0;
        ListNode evenDummy = new ListNode(-1);
        ListNode evenDummyTemp = evenDummy;
        ListNode oddDummy = new ListNode(-1,head);
        ListNode prev = oddDummy;
        while (oddTemp != null){
            count++;
            ListNode next = oddTemp.next;
            if(count % 2 == 0){
                oddTemp.next = null;
                evenDummyTemp.next = oddTemp;
                evenDummyTemp = evenDummyTemp.next;
                prev.next = next;
                oddTemp = next;
                continue;
            }
            oddTemp = next;
            prev = prev.next;
        }
        prev.next = evenDummy.next;
        return head;
    }
```
创建了一个新的链表，空间复杂度为O(n) 时间复杂度为O(n)

# 思路  空间复杂度为O1 时间复杂度On
也是分成奇偶链表，然后再组合，odd = head,那么even=head.next, 然后更新偶数点时，偶数节点链接奇数结点的下一个。更新奇数节点时，奇数结点的下一个链接到偶数结点的下一个。

```
public static ListNode oddEvenList(ListNode head){
        ListNode evenHead = head.next;
        ListNode odd = head;
        ListNode even = evenHead;
        while (even != null && even.next != null){
            odd.next = even.next;
            odd = odd.next;
            even.next = odd.next;
            even = even.next;
        }
        odd.next = evenHead;
        return head;
    }
```
