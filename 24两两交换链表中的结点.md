# 思路
默认题目给的事偶数链表，即进行链表结点的交换是关键。两个结点交换，也就是它们之间关系没有了，同时当前节点的前驱要连接到新的结点，当前结点的后驱的后驱也要被当前结点所连接。所以要有前驱随着链表遍历也被记录。


```
public ListNode swapTwo(ListNode head){
        ListNode cur = head;
        ListNode dummy = new ListNode(-1,head);
        ListNode prev = dummy;
        while (cur != null && cur.next != null){

            //交换
            prev.next = cur.next;
            cur.next = cur.next.next;
            prev.next.next = cur;

            //交换后cur到一下一个位置，而且进行下面的交换，即cur要再进行到下一个位置。因此prev先=cur,占据cur当前位置，然后cur倒下一个位置
            //仍然是prev.next = cur;
            prev = cur;
            cur = cur.next;
        }
        return dummy.next;
    }
```
