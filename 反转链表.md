# 思路
思路一：关键在于  要设置新的链表，就要一个一个的连起来，同时保证新连的链表的下一个结点不丢失，也就是遍历原链表的时候要先单独记录当前节点的下一节点，既然是反转链表，也就是头结点要变到尾，也就是依次的要被后面的结点所连接，而并非只是简单的他最后要连接一个空节点，
所以一样的思路，设计一个空节点所有的cur结点指向temp=null,然后temp再变成新的链表的头结点,然后再cur = cur.next，   这也就是一个简单的头插法的过程，然后原来的next正好也变成下一个要被插入的结点。  整体的思路就是 头插+ 遍历。


```
public ListNode reverse(ListNode head){
        //既然要反转，那么后面的结点最后要放到前面，也就是头插法
        ListNode cur = head;
        ListNode temp = null;
        while (cur != null){
            ListNode next = cur.next;
            cur.next = temp;
            temp = cur;
            cur = next;
        }
        return temp;

    }
```
