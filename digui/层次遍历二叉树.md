# 思路
同样使用队列，初始把根节点推进去，然后进入循环，循环继续条件为队列是不为空，再内嵌一个循环遍历当前层，从根节点开始就是1层，将根节点出队并记录打印它的值，左右孩子入队，此时队列里的循环就是二层结点的数量对应的循环，内嵌循环的作用就是遍历该层的每一个结点，  
并将该层的结点打印同时弹出并将该结点的左右孩子入队。 这层遍历完了也就是队列只剩下一层的结点也就是该层的孩子。如此反复即可。
```
public static void main(String[] args) {
        // Solution s = new Solution();
        Integer[] treeArray = {0,2,3,4,5,6};
        TreeNode root = buildTree(treeArray);
        System.out.println(root);

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()){
            int size = queue.size();
            System.out.println();
            for(;size > 0;){
                TreeNode cur = queue.poll();
                System.out.print(cur.val + "|");
                if(cur.left != null){
                    queue.offer(cur.left);
                }
                if(cur.right != null){
                    queue.offer(cur.right);
                }
                size--;
            }
        }
    }
    public static TreeNode buildTree(Integer[] treeArray){
        if(treeArray == null){
            return null;
        }
        TreeNode root = new TreeNode(treeArray[0]);
        int i = 1;
        Queue<TreeNode> queue = new LinkedList<TreeNode>();
        queue.offer(root);
        while (i < treeArray.length){
            TreeNode node = queue.poll();
            if(treeArray[i] != null){
                node.left = new TreeNode(treeArray[i]);
                queue.offer(node.left);
            }
            i++;
            if(i >= treeArray.length){
                break;
            }
            if(treeArray[i] != null){
                node.right = new TreeNode(treeArray[i]);
                queue.offer(node.right);
            }
            i++;
        }
        return root;
    }
```
