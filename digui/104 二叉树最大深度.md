# 思路
使用队列存储结点，每一层的结点数等于上一层结点的左右孩子的数量。即我们从root开始，单个结点，下一层也就是第二层的节点数字等于root节点的左右孩子数，然后每一层做一次遍历，寻找它们的结点的孩子数来判断下一层的结点，并且把下一层的结点推入队列。即获取到了当前节点的孩子就把他出队，然后加入它的孩子数即可。

```
public int maxDepth(TreeNode root) {
        if(root == null){
            return 0;
        }
        Queue<TreeNode> queue = new LinkedList<TreeNode>();
        queue.offer(root);
        TreeNode cur = root;
        int ans = 0;
        while (!queue.isEmpty()) {
            int size = queue.size();
            while (size > 0) {
                TreeNode node = queue.poll();
                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
                size--;
            }
            ans++;
        }
        return ans;
    }
```
# 递归
递归调用获取左右子树的最大深度+1(根节点的深度为1)
```
public int maxDepth(TreeNode root) {
        if(root == null){
            return 0;
        }
        int result = Math.max(maxDepth(root.left),maxDepth(root.right)) + 1;
        return result;
    }
```
